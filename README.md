# astrophysics-chatbot


## prerequisites

1. go to the official qdrant website, register an account, and create a new cluster. Create an api key in the API Keys tab in the cluster, and copy the url in the Use the API block. 

2. go to the official groq website, register an account, and go to API Keys and create an api key.


## steps 
1. Download the all the files in https://drive.google.com/drive/folders/1RS8kaxjfnsj3WJIotCE9PswN_rqCVX5v, and store them into a directory named `downloaded`. If running on colab, upload them (You may find ways to zip and unzip)

2. Fill in your qdrant url and api key, then run `embed.ipynb`

3. Fill in your qdrant url and api key, and groq api key, then run `astro_physics_chatbot.ipynb`
